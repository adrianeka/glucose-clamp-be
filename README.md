# About The Project

## Book Recipe - Studi Kasus Multi Teknologi

### Built With

* Java 17
* Springboot 3.1.7
* Maven 3.5.0
* PostgreSQL
* Other dependencies such as JWT, Lombok, MinIO

<!-- GETTING STARTED -->
### Getting Started

1. Clone the repo
   ```sh
   git clone https://gitlab.cloudias79.com/multi-tech-case-study/book-recipe/book-recipe-be.git
   ```
2. Install Maven packages
   ```sh
   mvn install
   ```
3. Run the application
   ```sh
   mvn spring-boot:run
   ```
4. Or build the project
   ```sh
   mvn clean package
   ```
